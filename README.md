# Ukraine 2025 Partner Distribution Data (Public Exports)

Auto-generated CSV exports from Direct Relief's Ukraine 2025 Partner Distribution Monitoring project. These files are intended for use in Felt and other tools that read data from public URLs.

## Files

### `ukraine_2025_unified_distributions.csv`

One row per partner-reported distribution line, combined across all partners. Regions are standardized to the `Felt_Name` format for geocoding.

Columns include: Partner, Submission_Status, Delivery_Order, Receiver, Receiver_Type, Region, Region_Felt, City, Product_Type, Quantity, Units, Notes, Region_ISO_Code.

### `ukraine_2025_completeness_metrics.csv`

One row per partner with reporting completeness metrics. Suitable for a partner-level summary widget.

Columns: Partner, Status, Shipped_Records, Shipped_Quantity, Shipped_Value_USD, Reported_Rows, Reported_Quantity, Coverage_Pct_Capped, Coverage_Pct_Raw, Unique_Regions_Reported, Unique_Receivers_Reported.
